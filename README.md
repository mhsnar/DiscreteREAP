## üìã **Input Fields Explained**


### **System Matrices**
1. **Matrix A**  
   Represents the state-transition matrix of the system.  
   **Example**:  
A = [0 1; -2 -3]

2. **Matrix B**  
Input matrix defining how the control inputs affect the system.  
**Example**:

B = [0; 1]

3. **Matrix C**  
Output matrix, mapping the state to system outputs.  
**Example**:  
C = [1 0]
